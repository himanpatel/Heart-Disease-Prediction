Welcome To eart Disease Prediction Project

Our Project focused on the predicting of heart disease using machine learning classification algorithms and later on we decided to build a prediction form to enable the doctors to add in patient details and predict if they have the heart disease or not.The team also developed an excellent interactive dashboard to display the graphs and show filters and colours as per requirements.
